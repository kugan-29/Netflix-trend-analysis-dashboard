## 📖 Overview

Streaming platforms release huge volumes of content every year. This project analyzes Netflix's content catalog to understand which genres dominate, how content mix has shifted over time, and which countries produce the most titles.

## ✨ Key Features

- Cleaned and analyzed a dataset of **8,000+ Netflix titles** using Python (Pandas)
- Uncovered **genre popularity trends** and release-year patterns
- Built **Excel pivot tables and charts** to visualize content growth by country
- Compared movies vs. TV shows distribution across regions and years

## 🛠️ Tech Stack

- **Python** (Pandas, Matplotlib) — data cleaning & analysis
- **Excel** — pivot tables, charts, and visualization
- **Jupyter Notebook** — analysis workflow

## 📊 Dataset

Netflix titles dataset (8,000+ records) including title, genre, country, release year, and content type. [Add dataset source link, e.g., Kaggle]

## 🚀 How to Run

1. Clone this repo: `git clone https://github.com/kugan-29/netflix-content-trend-analysis.git`
2. Install dependencies: `pip install pandas matplotlib jupyter`
3. Run the notebook: `jupyter notebook netflix_analysis.ipynb`


## 📈 Key Insights

- Identified top-performing genres by release volume
- Mapped how content production shifted across countries over the last 10 years
- Highlighted the growing share of international (non-US) content
